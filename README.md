## Contributing

Found an issue or want to improve a page? Please open a pull request. We review documentation changes regularly and appreciate your contributions.

## Local development

Install Mintlify CLI and start the dev server:

```bash
npm i -g mint
mint dev
mint broken-links
```

The site uses a root-level `docs.json` for navigation and settings. Content pages are `.mdx` files alongside it.
